# Runtime Interface Map

Canonical index of all public runtime interface repositories.

This repository contains no contracts, no schemas, and no logic.
It exists only to define scope separation and ownership boundaries.

## Public Interface Repositories

### Government
- gov-runtime-interface  
Public contracts for civic systems, PATH programs, and public infrastructure.

### Banking
- bank-runtime-interface  
Public contracts for banks, credit unions, and regulated financial institutions.

### Crypto
- crypto-runtime-interface  
Public contracts for blockchain, wallets, custody, and decentralized payments.

## Rules
- Interfaces only
- No execution logic
- No shared state
- No cross-contamination between domains

All implementations exist in private or sovereign runtimes.
